# Leotype 🚀

Современное приложение для тренировки скорости печати с крутыми анимациями и улучшенным UX. Вдохновлено Monkeytype, но с собственным уникальным дизайном и функциональностью.

## ✨ Возможности

- 🎯 **Различные режимы тестирования**
  - По времени (15, 30, 60, 120 секунд)
  - По количеству слов (10, 25, 50, 100)
  - Режим цитат
  
- 🎨 **5 красивых тем оформления**
  - Dark (темная)
  - Light (светлая)
  - Neon (неоновая)
  - Ocean (океан)
  - Forest (лес)

- ⚡ **Плавные анимации с Framer Motion**
  - Анимированные символы при вводе
  - Эффекты при правильных и неправильных символах
  - Конфетти при завершении теста
  - Фоновые эффекты во время теста
  - Анимированная статистика

- 📊 **Детальная статистика**
  - WPM (слов в минуту)
  - Точность в процентах
  - Время выполнения
  - Детализация по символам (правильные, ошибки, лишние, пропущенные)

- ⌨️ **Виртуальная клавиатура**
  - Подсветка активных клавиш
  - Визуальная обратная связь
  - Русская раскладка

- 🎭 **Современный интерфейс**
  - Адаптивный дизайн
  - Плавные переходы
  - Интуитивная навигация

## 🛠️ Технологии

- **React 18** - UI библиотека
- **TypeScript** - Типизация
- **Vite** - Сборщик
- **Framer Motion** - Анимации
- **Zustand** - Управление состоянием
- **Tailwind CSS** - Стилизация

## 📦 Установка

```bash
npm install
```

## 🚀 Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## 🏗️ Сборка

```bash
npm run build
```

- `npm run build` - Сборка для production

- `npm run format` - Форматирование кода

## 🎮 Использование

1. Выберите режим тестирования (время, слова или цитата)
2. Настройте параметры (время/количество слов, тему)
3. Начните печатать - тест запустится автоматически
4. После завершения просмотрите статистику
5. Используйте кнопку "Новый текст" для генерации нового текста
6. Используйте кнопку "Сброс" для перезапуска теста

## ⌨️ Горячие клавиши

- `Escape` - Остановить тест
- Любая клавиша - Начать тест
